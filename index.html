Carlos Andres Aristizabal Kenguan

Desarrollador web Full Stack


Soy un apasionado desarrollador web full stack con una sólida formación y con 3 años experiencia. mi enfoque se centra en crear soluciones tecnológicas eficientes y escalables que optimizan la experiencia del usuario y cumplan con los requisitos del negocio. Usando lenguajes de programación en Front-end como Html5, Css con el manejo del Back-end con JavaScript, Java, Python manejo de datos con MongoDB y MySql con sumiendos API's rest full y un control de veriones con Git y GitHub y proyectando el CI/CD con GitHub-Action, Docker y en la nube con AWS, y Jenkins y mucho más.



Sobre mi

Certificado como Developer Full Stack y titulado como Tecnólogo en Producción Multimedia, Técnico en Operador de Computo con Énfasis en Diseño Gráfico, Técnico laboral de Mantenimiento de Computo, en Electrónica y además certificado en varios Cursos en Coursera, LinkedIn learning y el SENA enfocados en mi campo.

A nivel personal, considero que asumo las responsabilidades que se me asignan con todo el compromiso, con la facilidad de adaptarme al cambio y al trabajo en equipo; como también, destaco en ser recursivo por explorar soluciones alternativas a los problemas y buscar constantemente mantenerme actualizado en todos los campos mencionados de mi interés. Mis competencias técnicas en los diferentes campos:

* Desarrollo Frontend: dominio de tecnologías como Html5, Css3, Javascript (es6+), React.js, creando interfaces de usuario atractivas y funcionales. * Desarrollo Backend: experiencia sólida en el diseño y desarrollo de servidores utilizando tecnologías como Node.js, Express.js, Python y Java. * Base de datos: profundo conocimiento en el diseño y optimización de bases de datos relacionales (Mysql, Postgresql) y no relacionales (MongoDB). * Apis: creación y consumo de Apis Restfull, diseño y documentación con Swagger ⁄ OpenApi. * Control de versiones: uso experto de sistemas de control de versiones como Git, Github. * Despliegue y gestión: experiencia en implementación y administración de servidores utilizando Docker y orquestación con Github Actions y Jenkins, conocimientos en AWS.
